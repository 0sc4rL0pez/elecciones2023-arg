To obtain information, I used mainly a  library called Instaloader 

/comentarios_iteradores:
There are iterators where the number of comments from each publication was collected (discarded)

/iteradoresV2:
Iterators are found to store information sequentially, for long periods of time.

/iteradoresV2_new:
Iterators are found to update information, for short periods of time. They have to be deleted once the update is finished.

/old_versions:
This is the first version of the main program
